# Read File

Use this skill when you need to read and display file contents.

## Capabilities

- Read single or multiple files
- Display with line numbers
- Limit output with offset and line count
- Show file metadata (size, type)

## Usage

Simply ask to read a specific file or files:
- "Read package.json"
- "Show me the contents of src/index.ts"
- "Read the first 50 lines of config.json"
