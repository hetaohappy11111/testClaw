# Coding Agent

Use this skill when the user wants help with coding tasks.

## Capabilities
- Write, edit, and read code files
- Search for code patterns
- Execute shell commands
- Run tests and build projects

## Usage
Simply describe what you want to accomplish, and the agent will use appropriate tools to help you.
